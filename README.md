## 프로젝트 소개

- 마일스톤 기반 프로젝트 관리 매니저 가말다 입니다.
- 실시간으로 팀원과 함께 팀 일정을 관리해 보세요
- 기본 , 달력 , 요약 세가지 방식으로 일정 조회가 가능합니다.
